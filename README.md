# GraduateProject

Для работы тестов необходимо установить библиотеки PyTest и Selenium, а также скачать драйвер для Selenium. Перед запуском укажите в файле Config телефон, почту и пароль существующего пользователя, а также адрес вашего вебдрайвера

Тестовые сценарии проверяют:
- test_auth - возможность авторизации на каждом из продуктов с помощью почты и телефона;
- test_auth_fields - наличие всех полей по требованиям в форме авторизации;
- test_timedcode_fields - наличие всех полей по требованиям в форме авторизации по временному коду;
- test_reg_fields - наличие всех полей по требованиям в форме регистрации.
